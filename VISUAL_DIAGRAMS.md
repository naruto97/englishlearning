# Visual Diagrams: Essay Structure

## Overview
These visual diagrams show how an argumentative essay is structured. Use them as mental maps while writing.

---

## DIAGRAM 2: Introduction Breakdown

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           INTRODUCTION                                   │
│                           (15 points)                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  SENTENCE 1-2: HOOK                                                     │
│  ┌────────────────────────────────────────────────────────────────────┐│
│  │ Choose ONE:                                                        ││
│  │ • Statistic: "3.7 million seniors face..."                         ││
│  │ • Question: "What if the choice at eighteen..."                    ││
│  │ • Scenario: "Every May, students stand at..."                      ││
│  │ • Bold Statement: "This is the most consequential..."              ││
│  └────────────────────────────────────────────────────────────────────┘│
│                              │                                          │
│                              ▼                                          │
│  SENTENCE 2-3: CONTEXT                                                  │
│  ┌────────────────────────────────────────────────────────────────────┐│
│  │ • Name the alternatives (college, trade school, work)              ││
│  │ • Explain why this matters                                         ││
│  │ • Acknowledge both sides exist                                     ││
│  └────────────────────────────────────────────────────────────────────┘│
│                              │                                          │
│                              ▼                                          │
│  SENTENCE 4: THESIS                                                     │
│  ┌────────────────────────────────────────────────────────────────────┐│
│  │ "Although [counterargument],                                       ││
│  │  [your position] because                                           ││
│  │  [reason 1], [reason 2], and [reason 3]."                          ││
│  └────────────────────────────────────────────────────────────────────┘│
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## DIAGRAM 3: Body Paragraph Structure (TEEL)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         BODY PARAGRAPH                                   │
│                        (TEEL Structure)                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌─ T ─────────────────────────────────────────────────────────────────┐
│  │ TOPIC SENTENCE                                                      │
│  │ • States ONE claim                                                  │
│  │ • Connects to thesis                                                │
│  │ • (¶2+: Includes transition from previous)                          │
│  └─────────────────────────────────────────────────────────────────────┘
│                              │                                          │
│                              ▼                                          │
│  ┌─ E ─────────────────────────────────────────────────────────────────┐
│  │ EVIDENCE                                                            │
│  │ • Specific data (numbers, percentages)                              │
│  │ • From credible source                                              │
│  │ • With proper MLA citation                                          │
│  │                                                                     │
│  │ Example: "According to BLS, graduates earn $1,432 weekly (BLS)."    │
│  └─────────────────────────────────────────────────────────────────────┘
│                              │                                          │
│                              ▼                                          │
│  ┌─ E ─────────────────────────────────────────────────────────────────┐
│  │ EXPLANATION                                                         │
│  │ • WHY the evidence matters                                          │
│  │ • Do the math for the reader                                        │
│  │ • Connect to real-world significance                                │
│  │                                                                     │
│  │ Example: "This $579 gap compounds to $1.2M over a career."          │
│  └─────────────────────────────────────────────────────────────────────┘
│                              │                                          │
│                              ▼                                          │
│  ┌─ L ─────────────────────────────────────────────────────────────────┐
│  │ LINK                                                                │
│  │ • Connect back to thesis claim                                      │
│  │ • OR transition to next paragraph                                   │
│  │                                                                     │
│  │ Example: "This return demonstrates college's value."                │
│  └─────────────────────────────────────────────────────────────────────┘
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## DIAGRAM 5: Citation Flow

```
                         ESSAY BODY
    ┌─────────────────────────────────────────────────────────────────┐
    │                                                                 │
    │  "Claim with data (Source)." ←── In-text citation              │
    │                     │                                           │
    │                     │                                           │
    │  "According to Source, claim with data." ←── Signal phrase      │
    │                     │                                           │
    └─────────────────────────────────────────────────────────────────┘
                          │
                          │ Every source cited in essay
                          │ MUST appear on Works Cited
                          ▼
                    WORKS CITED
    ┌─────────────────────────────────────────────────────────────────┐
    │                                                                 │
    │  Source. "Title." Website, Publisher, Date, URL. Accessed Date. │
    │                                                                 │
    └─────────────────────────────────────────────────────────────────┘
```

```
           │

           │

           │
           │  "While these advantages are substantial..."
           ▼
    ┌──────────────┐
    │   BODY ¶3    │

    └──────┬───────┘
           │
           │  "The evidence demonstrates..."
```

## DIAGRAM 7: Conclusion Requirements

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           CONCLUSION                                     │
│                           (15 points)                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  MUST INCLUDE:                           MUST NOT INCLUDE:              │
│  ┌────────────────────────────┐         ┌────────────────────────────┐ │
│  │ ✓ Restated thesis          │         │ ✗ New statistics           │ │
│  │   (different words)        │         │ ✗ New arguments            │ │
│  │                            │         │ ✗ New sources              │ │
│  │ ✓ Body ¶1 topic            │         │ ✗ New citations            │ │
│  │ ✓ Body ¶2 topic            │         │ ✗ "This is just my opinion"│ │
│  │ ✓ Body ¶3 topic            │         │ ✗ Undermining statements   │ │
│  │                            │         │                            │ │
│  │ ✓ Reflective element       │         │                            │ │
│  │ ✓ Final memorable thought  │         │                            │ │
│  └────────────────────────────┘         └────────────────────────────┘ │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
