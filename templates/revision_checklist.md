# Essay Revision Checklist

## Instructions
After completing your draft, go through this checklist systematically. Check each box when verified. Fix problems before moving to the next section.

---

## PASS 1: Structure Check

### Introduction (15 points)
- [ ] Has 4+ sentences (count them: _____)
- [ ] Sentence 1-2 is an engaging hook
- [ ] Hook is NOT "Many people..." or "This essay will..."
- [ ] Context explains the debate and alternatives
- [ ] Final sentence is the thesis
- [ ] Thesis states clear position
- [ ] Thesis previews 3 reasons

## PASS 2: Citation Check (15 points)

### Count Your Citations
- Body ¶1 citations: _____
- Body ¶2 citations: _____
- Body ¶3 citations: _____
- Total: _____ (aim for 4-6 minimum)

### Verify Format
- [ ] All citations are (Author) or (Organization)
- [ ] All citations are BEFORE the period
- [ ] No vague "Studies show" without citation
- [ ] Direct quotes have quotation marks AND citations
- [ ] No redundant citations (signal phrase + parenthetical)

**Fix needed:** _______________________________________________

---

## PASS 3: Works Cited Check (15 points)

- [ ] On separate page after conclusion
- [ ] Title "Works Cited" is centered, plain text (not bold)
- [ ] Everything is double-spaced
- [ ] Entries are in alphabetical order
- [ ] Each entry has hanging indent
- [ ] Every in-text citation has matching entry
- [ ] No entries for sources not cited

### Entry Format Check
For each entry, verify:
- [ ] Author/Organization. (period)
- [ ] "Article Title." (period inside quotes)
- [ ] *Website Name*, (comma)
- [ ] Publisher, (comma, if different from website)
- [ ] Date, (comma)
- [ ] URL. (period)
- [ ] Accessed Date. (period)

**Fix needed:** _______________________________________________

## PASS 4: Organization Check (15 points)

- [ ] Clear introduction (identifiable as beginning)
- [ ] Clear body (3 distinct paragraphs)
- [ ] Clear conclusion (identifiable as end)
- [ ] Paragraph order is logical
- [ ] Each transition is explicit (not implied)
- [ ] Each transition is strong (more than one word)
- [ ] Each transition is smooth (natural flow)
- [ ] No paragraphs jump between unrelated ideas
- [ ] Conclusion doesn't introduce new arguments

**Fix needed:** _______________________________________________

## PASS 5: Mechanics Check (10 points)

### Spelling
- [ ] Ran spell-check
- [ ] Checked homophones: their/there/they're
- [ ] Checked homophones: its/it's
- [ ] Checked homophones: your/you're
- [ ] Checked homophones: affect/effect
- [ ] Checked homophones: than/then

### Punctuation
- [ ] Commas after introductory phrases
- [ ] Commas before FANBOYS in compound sentences
- [ ] Semicolons between independent clauses (where used)
- [ ] Periods AFTER citations, not before

### Grammar
- [ ] Subject-verb agreement throughout
- [ ] Consistent verb tense (present for research)
- [ ] No sentence fragments
- [ ] No run-on sentences or comma splices

### Tone
- [ ] NO contractions (don't, can't, won't, etc.)
- [ ] NO first person (I, me, my, we, our)
- [ ] NO second person (you, your)
- [ ] Formal academic language throughout

### Sentence Variety
- [ ] Mix of simple, compound, and complex sentences
- [ ] Varied sentence lengths
- [ ] Varied sentence openings
