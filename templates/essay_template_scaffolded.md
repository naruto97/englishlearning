## PRE-SUBMISSION CHECKLIST

### Introduction
- [ ] 4+ sentences
- [ ] Engaging hook (not "Many people")
- [ ] Context explaining the debate
- [ ] Clear thesis with position
- [ ] Three reasons previewed in thesis

### Body Paragraphs
- [ ] THREE body paragraphs
- [ ] Each paragraph has 4+ sentences
- [ ] Each paragraph has ONE claim
- [ ] Each claim has evidence WITH citation
- [ ] Each piece of evidence is explained
- [ ] Transitions between paragraphs

### Conclusion
- [ ] 4+ sentences
- [ ] Thesis restated in NEW words
- [ ] ALL THREE points mentioned
- [ ] No new information
- [ ] Memorable final thought

### Works Cited
- [ ] On separate page
- [ ] "Works Cited" centered, plain text
- [ ] Alphabetical order
- [ ] Hanging indent
- [ ] All sources cited in essay are listed

### Mechanics
- [ ] No contractions (don't → do not)
- [ ] No first person (I, me, my)
- [ ] No second person (you, your)
- [ ] Varied sentence structures
